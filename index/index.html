Digilent

In 2016, I was an intern at Digilent Inc in Pullman, Washington. Digilent provides technology-based educational tools to teach electrical engineering concepts to students and hobbyists. Among their products are microprocessor and microprocessor modules. These mini computers can do many functions, from measuring pressure and temperature, to connecting to bluetooth, to hosting videogames. I helped code libraries for their line of microprocessor modules, making it easier for users to get started with their projects. I also coded many exampled demos for these products. My favorite demo was a high five machine, which would use a motor to spin a robot arm, and a light sensor to detect when someone was high fiving it. I also wrote documentation and blog posts to help educate and sell their products. It was a rewarding experience to provide people the tools to educate themselves on engineering topics.

Twist Bioscience

Twist Bioscience is a San Francisco based company that manufactures synthetic DNA for the biotechnology industry. Their synthetic DNA has many applications, from detecting early cancer tumors, to storing OK GO music videos in DNA memory. In 2015, I got an offer to help Twist Bioscience visualize some of their lab data. Previously, their vapor disposition lab data was stuck in a LabVIEW program which didn't provide a lot of functionality for visualizing data. I helped by writing a Python script that grabbed this labVIEW data and stored the raw data into an Excel document. It would also analyze the data using a Python data analysis library called Panda, and look for several things such as sudden changes in pressure or temperature, or the mean difference between lab tests. It was really exciting to work with Biotech scientists and help analyze their data.

Tools I Used

  • LabVIEW
  • Python
  • Panda Data Analysis Library

Polymorphic Games

One of my proudest achievements at the University of Idaho was working with Polymorphic Games. Polymorphic Games is a programming studio populated with undergraduate students, who were tasked with creating educational videogames that teach the foundations of evolutionary biology. Using genetic algorithms to program enemy AI, we were able to create a game that evolves based on the players actions. For example, if the player failed to kill enemies that were fast and had fire attacks, these traits would spread to the next wave of enemies, effectively making the game harder. This project helped me learn how to program AI, to develop software with other engineers, and to work with biologists and artists to create something great.

Tools I Used

  • C#
  • Unity
  • Github
